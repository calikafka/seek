## Words

+ cURL 
++ The command line web browser! 
++ curl simplifies data collection from websites via its command-line interface. You can use curl in your terminal to directly ask a server for information, like a web page or file, without needing a browser.
++ This works for simple websites, not JavaScript-heavy modern websites.
++ Standard cURL cannot execute JavaScript on its own; it only retrieves the raw, static HTML "skeleton" from a server
++ cURL is a lightweight, command-line tool ideal for scraping static pages and structured JSON data.
